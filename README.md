## Build instructions:

1. Clone the code.

2. Go to project folder.

3. To use default **appsettings.json** (reports will be saved in the project folder) run:

```shell
dotnet run
```

4. To use custom args run:

```shell
dotnet run --IntervalMunites 1 --FolderPath "C:\Users\<YOUR_USER>\Desktop\"
```
